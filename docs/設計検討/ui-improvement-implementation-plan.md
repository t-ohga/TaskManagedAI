## セキュリティ契約 (全データ境界)
1. **Server Actions**: 認証→認可→スコープ→遷移検証→冪等性
2. **Route Handlers / Loaders**: tenant/project スコープ必須
3. **検索/フィルター**: 入力エスケープ + パラメータ化
4. **Markdown**: sanitize + 許可タグリスト + XSS 回帰テスト
5. **Export/Import** (Tier 4): design approval 必須
6. **API キー管理** (Tier 4): design approval 必須
