TaskManagedAI を「Linear / Notion / GitHub Projects と同等の操作性を持つ
AI-native タスク管理ツール」にする。

- Claude 独自分析: 全 18 ページの機能欠落洗い出し (30+ 項目)

## 実装 Units (UX インパクト × 実装難易度 で優先順位付け)

### Unit 1: チケット操作の完全化 [最優先]
- **ステータス変更ボタン** (チケット詳細から1クリックで変更)
- **インライン編集** (タイトル/説明をダブルクリックで編集)
- **チケット削除** (確認ダイアログ付き)
- 難易度: MEDIUM / インパクト: HIGH

### Unit 2: 検索・フィルターシステム
- **グローバル検索バー** (ヘッダーに常設、Cmd+K でフォーカス)
- **ステータスフィルター** (看板ページにドロップダウン)
- **優先度フィルター**
- **日付範囲フィルター**
- 難易度: MEDIUM / インパクト: HIGH

### Unit 3: ビュー切替 (看板 / リスト)
- **リスト表示モード** (テーブル形式、ソート可能)
- **ビュー切替ボタン** (看板 ⇄ リスト)
- **ソート** (作成日/優先度/ステータス/タイトル)
- 難易度: MEDIUM / インパクト: HIGH

### Unit 4: データ可視化
- **ステータス分布チャート** (ダッシュボードに円グラフ)
- **完了率プログレスバー** (プロジェクトカードに)
- **日次トレンド** (簡易バーチャート)
- 難易度: MEDIUM / インパクト: MEDIUM

### Unit 5: トースト通知 + ローディング UX
- **トースト通知** (操作成功/失敗のフィードバック)
- **ローディングスケルトン** (カード形状のプレースホルダー)
- **楽観的更新** (ステータス変更の即時 UI 反映)
- 難易度: LOW / インパクト: HIGH

### Unit 6: ナビゲーション強化
- **パンくずリスト** (全ページに統一)
- **グローバル検索** (Cmd+K コマンドパレット)
- **最近閲覧したチケット** (ヘッダーに表示)
- **ナビゲーション active 状態** (現在ページのハイライト)
- 難易度: MEDIUM / インパクト: MEDIUM

### Unit 7: ダークモード
- **CSS 変数ベースのダークテーマ**
- **トグルスイッチ** (ヘッダーに)
- **システム設定との連動**
- 難易度: MEDIUM / インパクト: MEDIUM

### Unit 8: コメント・アクティビティログ
- **チケットコメントフォーム** (チケット詳細に)
- **アクティビティタイムライン** (誰が何をしたか)
- **Markdown レンダリング**
- 難易度: MEDIUM / インパクト: HIGH

### Unit 9: モバイルレスポンシブ
- **看板の縦スクロール** (モバイルでは横カラム → 縦リスト)
- **ハンバーガーメニュー** (ナビゲーション折り畳み)
- **タッチ操作最適化**
- 難易度: MEDIUM / インパクト: MEDIUM

### Unit 10: Codex レビュー + 品質最終確認
- 全変更の Codex adversarial review
- TypeScript ビルド確認
- Docker 再ビルド
- E2E テスト追加
